# Online Retail II Dataset - Exploratory Data Analysis (EDA)

## Project Overview

This project presents an Exploratory Data Analysis (EDA) of the **Online Retail II Dataset**. The objective is to analyze customer purchasing behavior, identify data quality issues, and generate business insights that can help the company make better business decisions before implementing a recommendation system.

## Dataset

* **Dataset:** Online Retail II
* **Source:** UCI Machine Learning Repository
* **Duration:** 2009–2011
* **Data Includes:**

  * Invoice Number
  * Product Code
  * Product Description
  * Quantity
  * Invoice Date
  * Unit Price
  * Customer ID
  * Country

## Objectives

* Load and understand the dataset.
* Check the dataset shape, columns, and data types.
* Identify missing values and duplicate records.
* Analyze the top 10 best-selling products by quantity.
* Analyze the top 10 products by revenue.
* Perform country-wise sales analysis.
* Visualize monthly revenue trends.
* Create a correlation heatmap.
* Detect outliers using boxplots.
* Generate business insights from the analysis.

## Business Insights

* A small number of products generate a significant portion of total sales.
* The United Kingdom contributes the highest revenue.
* Monthly sales fluctuate, indicating seasonal demand.
* Missing Customer IDs suggest anonymous purchases.
* Outliers indicate bulk purchases and product returns.

## Conclusion

The analysis provides valuable insights into customer purchasing behavior, product performance, and sales trends. These findings can support inventory planning, marketing strategies, and future recommendation system development.
